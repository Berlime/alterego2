## Prerequisites

1. Create new instance using the template.
	1. While waiting, update your WordPress website.
	2. Create a full back-up
	3. Download the back-up file to your computer
2. Point your domain to the server that have Dokploy.
3. Change domain name in dokploy
	1. Straight away use SSL
	2. Then click deploy
	3. Once u see the WordPress page, change cloudflare settings to turn on the proxy
	4. Install and activate WP Vivid plug-in
	5. Upload downloaded zip files
	6. Restore
